## 第一步：找到你的 VPS 连接信息

付款完成后搬瓦工会发一封邮件，里面有 root 密码。IP 地址和 SSH 端口在 KiwiVM 控制面板里查。

登录搬瓦工账户后，进 Services → My Services，找到你买的套餐，点"KiwiVM Control Panel"。进去之后首页就能看到：

- **Public IP address**：你的 VPS 公网 IP
- **SSH Port**：SSH 连接端口（搬瓦工不用默认 22，是随机生成的五位数）
- **Root Password**：首页没有，去"Root password modification"重置一个新的，重置完只出现一次，记下来

就这三样东西，后面全靠它们。

---

## 第二步：连上 SSH

拿到 IP、端口、密码，就可以连了。

**Windows**：用 Xshell（免费版够用）或者 Putty，新建连接填 IP 和端口，用户名 root，密码就是上面的 root 密码。

**Mac**：直接开终端，输这行命令：

bash
ssh root@你的IP地址 -p 你的SSH端口


回车，输密码，搞定。

**Linux**：终端里同样的命令，没啥区别。

第一次连会弹一个确认提示，选 yes 或者保存，后面就不再问了。连接成功之后会看到命令行提示符，到这一步 VPS 就算真正到手了。

顺便提一句：连接后建议立刻把 root 密码改掉，改成你自己记得住又复杂的。命令就一行：

bash
passwd

## 第三步：检查机房，要不要迁移

买了搬瓦工，有个不太明显但很重要的操作——看一下自己现在用的是哪个机房，如果不合适可以免费迁。

KiwiVM 里找"Migrate to another DC"，里面会列出当前套餐可以切换的所有机房。CN2 GIA-E 套餐支持超过 12 个机房随时切换，这是搬瓦工比较有意思的地方。

怎么选机房？按运营商来：

- **电信用户**：DC6 CN2 GIA-E 或 DC9 CN2 GIA，都走 CN2 GIA 回程，效果最好
- **联通用户**：荷兰 EUNL_9（联通 9929 线路），或者 CN2 GIA 机房也行
- **移动用户**：日本大阪软银 JPOS_1，移动去程走 CMI，体感顺畅

不确定的话可以先测一下延迟。搬瓦工各机房都有测速 IP，ping 一下哪个低就选哪个。

迁机房是免费的，但有次数限制，一般不要无限折腾。选好之后迁一次，安心用就行。

## KiwiVM 控制面板有哪些功能值得知道

KiwiVM 是搬瓦工自研的控制面板，是它和其他 VPS 商家比较明显的差异之一。几个常用功能：

- **一键重装系统**：Reinstall OS，选个你喜欢的 Linux 发行版，几分钟搞定。重装完会发邮件给你新的 root 密码
- **快照**：Snapshot，相当于存档，重装之前先打个快照，不满意可以恢复
- **迁移机房**：Migrate to another DC，上面说过了
- **流量统计**：能看到当月用了多少流量，快到限额的时候注意一下
- **重置密码**：Root password modification，密码忘了从这里重置

控制面板是英文的，但结构比较清晰，用几次就熟了。

## 几个常见问题

**Q：付款完成了但 VPS 没有开通怎么办？**

A：一般 5-10 分钟内会自动部署，邮件会发通知。如果超过半小时没动静，去搬瓦工后台提工单，客服响应速度还行，提交后通常不用等太久。

**Q：SSH 连不上怎么排查？**

A：第一步确认 VPS 是否在运行状态（KiwiVM 里看 Status）；第二步确认 IP 和端口对没对，端口搬瓦工不是默认 22，要去 KiwiVM 查准确的数字；第三步检查本地网络有没有问题。IP 被墙是另一种情况，可以在 KiwiVM 里申请换 IP（收费，$8 一次）。

**Q：套餐选错了买了 KVM 想换 CN2 GIA-E，能升级吗？**

A：直接升级的话比较麻烦，一般是新买一个套餐。如果还在 30 天内，可以先退款再重新选。30 天退款是支持的，操作在后台就能完成。

**Q：限量版套餐是什么，真的值得等吗？**

A：限量版套餐性价比比常规套餐高很多，同配置价格可能只有常规的一半甚至更低。但库存极少，补货后几小时内就没了，需要主动盯着。搬瓦工官方有邮件通知，可以在账户里设置。

**Q：流量超了怎么办？**

A：超出月流量限额会被限速，不是断线。等到下个月重置周期自动恢复，或者可以额外购买流量包。用量大的话选流量多的套餐，CN2 GIA-E 高配版月流量可以选到几 TB。
